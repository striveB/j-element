Icon 图标 ​

提供了一套常用的图标集合。

使用方法 ​

直接通过设置类名为 el-icon-iconName 来使用即可。例如：

Show Code
vue
<template>
  <i class="j-icon-edit"></i>
  <i class="j-icon-share"></i>
  <i class="j-icon-delete"></i>
  <j-button type="primary" icon="j-icon-search">搜索</j-button>
</template>
